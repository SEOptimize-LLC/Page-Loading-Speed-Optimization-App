Caching & Performance Plugins

  • Install a caching plugin such as WP Rocket, LiteSpeed Cache, or W3 Total Cache.
  • Enable page caching, browser caching, and GZIP/Brotli compression.
  • Use an image optimization plugin like ShortPixel or Imagify.

Theme & Plugin Audit

  • Disable or remove unused plugins to reduce JavaScript and CSS bloat.
  • Choose a lightweight, performance-oriented theme (GeneratePress, Astra, Kadence).
  • Avoid page builders that inject excessive inline CSS and JS.

Server & Hosting

  • Use a managed WordPress host with built-in caching (Cloudways, Kinsta, WP Engine).
  • Enable a CDN such as Cloudflare, BunnyCDN, or your host's built-in CDN.
  • Ensure PHP 8.1+ is enabled for improved server-side performance.
{% elif cms_name == "shopify" %}

Theme Optimization

  • Use a speed-optimized Shopify theme (Dawn, Starter, or similar OS 2.0 themes).
  • Minimize the number of Liquid render blocks and nested loops.
  • Lazy load product images using the loading="lazy" attribute.

App Audit

  • Audit installed apps and remove any that inject scripts you no longer use.
  • Check for apps that load JavaScript on every page even if only used on specific pages.
  • Prefer apps that use Shopify App Embed blocks over script tag injection.

Liquid & Asset Optimization

  • Use {% raw %}{{ 'file.js' | asset_url | script_tag }}{% endraw %} with defer for non-critical scripts.
  • Minimize Liquid render time by reducing complex conditional logic.
  • Enable Shopify's built-in CDN and ensure images use the image_url filter with size parameters.
